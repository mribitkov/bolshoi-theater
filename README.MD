# Bolshoi Theater

![Preview](https://github.com/mribitkov/bolshoi-theater/blob/docs/preview.png)

Fixed layout of the Bolshoi Theater site (1920px). Made on the basis of a template from the [Figma.info](https://figma.info/) website. The site is made on HTML5, CSS, and also uses the Normalize.css library. Designed by [Denis Timoshitsky](https://d-e-n.ru/).
